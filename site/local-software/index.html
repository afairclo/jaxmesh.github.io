Local Software

Is it software, but not for a server? For you, your phone or your computer? You probably need some knowledge on how to run/use these things.

  • Meshtastic-MQTT-Connect by pdxlocations - A python (TK/TCL) based MQTT/Meshtastic UI for your local machine.
  • MultiMesh - A cross-platform Meshtastic client based on Flutter
  • Meshtastic-SDR - A GnuRadio and Python based replacement for Meshtastic radio hardware. Can receive multiple presets at the same time. Currently working on TX.
  • Meshtastic-SAME-EAS-Alerter by RCGV1 - A tool to recieve alerts from the National Weather Service and broadcast them on Meshtastic
  • meshtastic-go by lmatte7 - A meshtastic CLI written in GoLang
  • Meshtastic python scripts by N7IW - Various Meshtastic python scripts for experimentation
  • Meshlink - Send messages and packet info to and from Discord
  • MeshTenna - Windows/Android app for comparing the performance of different antennas and locations
  • MeshSense - Windows/Mac/Linux app for using a node over Wi-Fi or Bluetooth
  • Meshtastic Visualizer - Python PyQT desktop app to interface with local nodes, or subscribing to MQTT servers
  • Meshtastic Plugin for Pidgin - A plugin to enable you to use Meshtastic with Pidgin (a multiplatform chat client)
